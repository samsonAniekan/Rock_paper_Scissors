## This project just showcases a rock , paper , scissors game where a user comes up against a computer
the computer value is selected at random based on the code and the user has to decide its selection for either of Rock, paper, scissors
The conditions for winning are simple rock beats scissors, paper beats rock, while scissors beats Paper.so the winner emerges from these choices
There would be further inputs into the game to make it something anyone with their phones can simply visit the webpage and just start playing the game
